Z61 MDIS Driver Documentation

This is the documentation of the MDIS low-level driver for the 16Z061_PWM FPGA module.

The 16Z061_PWM is a PWM generator implemented in an Chameleon FPGA. It supports the following features:

  • 16bit Register access to PERIOD and PULSE register


Variants

( no variants )

Functional Description


General

No hardware initialization is needed to be done by the driver.


Reading the register

The application can access the PERIOD and PULSE register by GetStat for reading and SetStat for writing data.


Descriptor Entries

The low-level driver initialization routine decodes the following entries ("keys") in addition to the general descriptor keys:

Descriptor entry Description Values

Z061_INIT_PERIOD Initial value for PERIOD register 0..0xFFFF, default: 0
Z061_INIT_PULSE Initial value for PULSE register 0..0xFFFF, default: 0


Interrupts

The Z061 PWM module doesn't generate any interrupts, so the driver doesn't support interrupt/signal handling.
